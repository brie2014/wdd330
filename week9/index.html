Notes

  • window.location.reload() forces page to reload
  • if given parameter of true it will reload form server, not cached page
  • window.history gives info about previously loaded pages in current browser session
  • window.open() and window.close() used to create popups; not usually best to use now
  • window.screen gives info about screen size and other properties; better to use media queries
  • create a cookie using document.cookie and adding a property and value which creates a special string
  • document.cookie = 'name=Superman'
  • you can add more properties that are appended to the end of the string
  • document.cookie = 'city=Metropolis' --> cookie now has two properties
  • you can change the value of a cookie's property by using the same name but giving a different value
  • document.cookie = 'name=Batman'
  • web workers use messages to communicate back and forth between main.js and worker script
  • worker scripts let script run in a separate thread without stopping the flow of the main script
  • audio and video elements have properties and methods to control playback
  • play(); pause(); volume=number; muted=bool; currentTime += number (to skip); playbackRate = number (fast forward or rewind); loop = bool; duration;
  • canvas element used for drawing
