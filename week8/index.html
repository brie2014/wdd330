Notes

  • transform property in CSS lets you translate, rotate, scale, or skew any element
  • translation: move elements left, right, up, or down
  • syntax example: transform: translate(45px, -45px)
  • scaling: lets you grow or shrink element w/ same ratio
  • syntax example: transform: scale(2); doubles the size
  • rotation: lets you rotate element around point of origin
  • syntax example: transform: rotate(-90deg);
  • skew: specifies skew along x and y axis
  • syntax example: transform: skew(15deg, 4deg)
  • transition lets property change gradually over time
  • transition-property--choose what property(s) to transition
  • transition-duration--how long transition will take
  • transition-timing-function--further manages pace of transition (ease-in, ease-in-out, linear, etc.)
  • transition-delay--wait time before transition begins; helps make sure user is intentionally causing transition
  • transition shorthand--combines all; ex {transition: color 0.2s ease-out 50ms;}
  • animation can be done with keyframes
  • create keyframes, then add animation properties
