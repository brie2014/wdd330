Week 2 Practice and Notes

Practice

  • Calculator Team Project
  • Hero Quiz Activity (Rough Draft)

Notes

  • there are different ways to create functions: declaration, expression, and arrow
  • use loops to go through an array
  • callback functions are useful when you want to use the backbone of one function to send it to others
  • you can use a callback function when making a calculator app
  • make sure you convert your data into the type you need it as (ex numbers from user inputs might need to be parsed into numbers)
  • DRY: Don't Repeat Yourself--get the basics, then go add fancier features; edit to see what you can condense

Questions

  • Look more into callbacks to firm up understanding--basically seems like a function calling a function
  • How do you correctly use event listeners to call functions? That was my biggest issue this week.
