Notes

  • use console.log to help check if you are sending/receiving/selecting what you think you are
  • forms have a default setting to reset when submitted that you can prevent from running
  • Conditional (ternary) operators
  • ternary operators are kind of like a shortcut for if statements
  • made up of condition, truthy expression, and falsy expresion
  • pressing on a key triggers a keydown event--each key has a data-key
  • keyboard data keys can be used to trigger events in functions!
  • local storage keys and values can be viewed in the browser inspector by clicking on applications, then local storage
  • JSON.stringify turns objects into strings (useful for local storage)
  • JSON.parse turns strings back into objects (useful for local storage)

Questions

  • Can you send an array of arguments through an if else statement? With the tic-tac-toe game we did an else statement for each winning possibility. Could we store all those combos, then loop through them?
