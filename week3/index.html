Notes

  • Arry Methods: .join & .push basics
  • .push and can be used to "push" new values into an array
  • .join can be used to join the elements of an array into a string
  • .push and .join can actually be used along with DOM to add elements to an HTML document
  • Arrary Iterations Cheat Sheet: forEach, filter, map, reduce, some, every, find, findIndex
  • forEach: basically loops through an array
  • filter: keeps only items in array that meet criteria
  • map: creates new array of list items
  • reduce: returns value of last iteration through; can be used to find totals or occurences of a value
  • sort: sorts items/objects in array by desired value
  • create const when you select something with DOM for reusability of code; that way you don't have to rewrite each DOM selection in the functions, just change the constants
